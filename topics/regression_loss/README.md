# 회귀 loss의 기하학적 직관 (hub)

> 본문 §2.2의 회귀 loss(MSE/MAE/Huber) 부분을 깊이 풀어 쓴 hub. 본문은 정의·핵심만 두고 *왜·어떻게 시각화되는가*는 여기서.

---

## 무엇을 다루는가

회귀에서 *loss를 그림으로 보는 법*. 정의식만 외우면 와닿지 않지만, 산점도 위에 잔차를 그리고 그 잔차의 *길이 / 제곱 / 절댓값*이 무엇을 뜻하는지 시각으로 잡으면 다음이 명확해진다.

- *왜 MSE는 outlier에 끌려가는가*
- *왜 MAE는 robust하지만 부드럽지 못한가*
- *왜 Huber가 둘의 절충인가*
